# Food Delivery Web Application

A full-stack Food Delivery application built with the MERN stack (MongoDB, Express.js, React.js, Node.js).

## Features
- User authentication (Login/Register)
- Browse and filter food items by category
- Cart management
- Order placement with Stripe payment integration
- Admin panel for food and order management
- Fully responsive design

## Tech Stack
- **Frontend:** React.js (Vite)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Payments:** Stripe
- **File Uploads:** Multer

## Getting Started
See the `.env.example` files in each directory for required environment variables.
